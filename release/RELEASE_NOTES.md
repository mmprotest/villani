# Villani 1.0.0

Villani 1.0.0 is the self-service daily-driver Pro release. It adds guided
repository and coding-system setup, verified stable/beta/pinned updates with
rollback, comprehensive local diagnostics, privacy-preserving support bundles,
offline entitlements, branch and pull-request delivery, Activity repeat, and
proof-summary sharing.

No update is forced. Update checks send only the installed Villani version and
never send repository source, prompts, diffs, or repository identity.
